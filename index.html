DeepAlgebra is an open-source AI project with a goal to automate formalization of mathematics and theorem proving. In particular we are interested in teaching machines to prove entirely new theorems, not proved before by humans. We concentrate on algebra and algebraic geometry.

Plan:

The technical background for what we do is explained here:

DeepAlgebra - an outline of a program, Przemyslaw Chojecki

We currently want to implement a dictionary between (La)Tex and COQ using syntactic parsers, and test it on The Stacks Project. The code will eventually appear on this website.

Technical plan:

  • Build a dictionary by using SyntaxNet or a similar syntactic parser to identify Types and variables in a mathematical text. Transfer those to COQ using a parse tree and a rule-based approach machine translation for environments and plain English.
  • Test this dictionary on the Stacks Project (a repository of facts from algebraic geometry).
  • Proceed to verification and theorem proving using E in COQ and hammers (a deep learning method).
